---
name: presentation-order-picker
description: Generate a random presentation order for a group. Use when a team needs a fair speaking sequence.
---

## When to use
Use when a group needs to decide a fair presentation or speaking order

## When not to use
Do not use when the order must follow a specific rule or priority

## Input
A list of participant names

## Output
A randomized presentation order list

## Steps
1. Take the list of names
2. Use the script to shuffle the order randomly
3. Return the ordered list

## Limitations
The result is random and may change each time
