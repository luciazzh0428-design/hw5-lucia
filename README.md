# Presentation Order Picker Skill

## What
This skill generates a random presentation order for a group.

## Why
In many team projects, deciding the speaking order fairly can be difficult. A random process ensures fairness. This task requires code because randomness cannot be reliably handled by a language model alone.

## How to use
Provide a list of names, and the skill will return a randomized order.

## Script
The Python script uses the random.shuffle() function to generate a fair random sequence.

## What worked well
The skill is simple, reusable, and directly applicable to real group projects.

## Limitations
The output is random and does not consider preferences or constraints.
